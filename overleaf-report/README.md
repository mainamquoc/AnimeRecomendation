# Overleaf import package

Upload the complete contents of this folder to one Overleaf project. Set the compiler to **XeLaTeX** and the bibliography tool to **Biber**, then compile `main.tex` twice if references do not appear on the first run.

The project is self-contained: all referenced figures are in `figures/` and all references are in `references.bib`. The report uses Times New Roman when it is available; otherwise it automatically uses the Overleaf-safe, Times-compatible TeX Gyre Termes font.

Before submission, confirm the contribution percentages match the group presentation and replace them if the members agree on a different split.
